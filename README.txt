ReadMe
=========

这个git用来记录我对数值分析这门课的想法，并希望能帮助到学习这门课程有困难的人，要维护这个项目得花费我大量的时间，这对我对这门课、对数值计算方法的理解和掌握程度是大有好处的，我会一直维护这个项目直到这门课结束。

主要会包含这些东西：

  - 一些感兴趣的证明
  - 每次的习题
  - 各种算法的实现，以及对比
  - 其他感兴趣的东西，自己经常用的数值计算方面的工具
